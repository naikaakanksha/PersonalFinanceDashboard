## Project Overview
The **Personal Finance Dashboard** is a web application built using **Python and Flask** to help users efficiently **track, manage, and analyze their personal finances**.  

This application enables users to:  
- Register and log in securely  
- Add, edit, and delete income and expense transactions  
- View financial status through an interactive dashboard  
- Analyze spending patterns across different categories  

This project highlights skills in:  
- **Backend Development:** Handling requests, processing data, managing user sessions  
- **Database Integration:** Connecting Python with MySQL to store and manage data  
- **Data Visualization:** Interactive charts for financial insights  
- **Frontend Development:** User-friendly and responsive interface  

Overall, it simulates a **real-world finance application**, making it recruiter-friendly and easy to understand.

---

## Features
- Secure **user authentication**  
- **Transaction management** (add/edit/delete income & expenses)  
- Interactive **dashboard charts**  
- **Category-wise analysis** of spending  
- Responsive and mobile-friendly design  
- MySQL database for secure data storage
